API Documentation

Integrate FetchXRates into your application

Authentication

All API requests require authentication using your API key in the header:

Authorization: Bearer {{ apiKey }}

Get Latest Rates

GET https://api.fetchxrates.com/v1/latest

Retrieve the latest exchange rates

{{ '{' }}
  "base": "USD",
  "date": "2026-02-07",
  "rates": {{ '{' }}
    "EUR": 0.92,
    "GBP": 0.79,
    "JPY": 148.50,
    "AUD": 1.56
  {{ '}' }}
{{ '}' }}

Get Historical Rates

GET https://api.fetchxrates.com/v1/historical?date=2026-01-01

Retrieve exchange rates for a specific date

Convert Currency

GET https://api.fetchxrates.com/v1/convert?from=USD&to=EUR&amount=100

Convert an amount from one currency to another

{{ '{' }}
  "from": "USD",
  "to": "EUR",
  "amount": 100,
  "result": 92.00,
  "rate": 0.92,
  "date": "2026-02-07"
{{ '}' }}
